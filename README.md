# qa-lab-01
# Objetivo

El objetivo de este ejercicio es aprender a crear funciones simples en Python y validarlas mediante pruebas automatizadas utilizando `pytest`.  
Se busca comprender cómo la verificación automática ayuda a asegurar que el código funcione correctamente según las reglas definidas.

---

# Herramientas utilizadas

- Python 3.x  
- pytest  
- Visual Studio Code  
- Git y GitHub  

---

# Ejecución de pruebas con pytest

Para ejecutar las pruebas se utilizó el siguiente comando en la terminal:
